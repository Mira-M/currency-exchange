Welcome

Direct Exchange is a simple web site that provides free currency exchange rates directly from your browser, cell phone, or tablet! Direct Exchange also provides a history for many countries dating back to 1999! On the main page, click either the Conversion link or the History link to get started.

Conversions

Direct Exchange's user friendly design, allows users to choose from a drop down or type in a country code to get started. As you type in your selections, a helpful list of supported countries will populate based on what you are typing. Simply type in or choose the country that you would like to convert from (base country).

After selecting the base country, choose the country that the currency will be converted to, from the Destination country drop down list or simply type in a country code.

Once you have selected a base and destination country, click the Submit button to see your conversion! It's that simple!

History

If you click on the History link, you can get the percent change in one country's currency from the selected date to the most current rate! Select a desired data between the current available date and 01/01/1999. Please note that some countries will not have history back to that date, the app will give you a friendly error if that rate is not available for that date. Once on the History page, click the calendar icon and choose a date.

Choose a country by either typing in the country code (again, helpful hints available), or choose a country from the drop down.

Once both the country and a valid date is selected, you can click submit to see the conversion from that date!

System Requirements

Direct Exchange can be accessed from any device that has a web browser and an internet connection. An Internet connection is required to use this application and standard data rates may apply on mobile devices that are not connected to wifi.
